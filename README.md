# Claude Code Repository

## 📌 Project Overview
The Claude Code project is a comprehensive toolchain for CLAUDE 4.X development, featuring:
- Custom prompt engineering environment
- Novel tool integration framework
- Specialized agent modeling agents
- Environment configuration utilities
- Deep system call monitoring capabilities

## ⚙️ Setup Instructions
1. **Prerequisites**
   - Python 3.9+
   - Docker Engine 20.0+
   - CLI environments (Windows/Mac/Linux)

2. **Environment Configuration**
```json
{
  "env": {
    "ANTHROPIC_BASE_URL": "http://localhost:4000",
    "ANTHROPIC_AUTH_TOKEN": "freecc"
  }
}
```
   Add the above to your `.claude/settings.json` under the `env` section.

3. **Install Dependencies**
```bash
pip install -r requirements.txt
```

4. **Run the CLI**
```bash
claude code start
```

## 🚀 Usage Examples
```bash
# Start a new planning session
claude plan "Add feature X"
```
